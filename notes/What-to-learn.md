# What to learn

## List with books:
1. Algorithms
    - Grokking Algorithms by Aditya Bhargava.
    - Introduction to Algorithms, by Thomas H. Cormen, Charles E. Leiserson, Ronald L. Rivest, Clifford Stein
2. Data
   - Designing Data-Intensive Applications by Martin Kleppman
   - Learning SQL by Alan Beaulieu (Free Ebook)
3. Clean Code
4. SOLID Principles
5. OO Programming
6. Design Patterns
7. Distributed Computing/systems
    -  Understanding Distributed Systems by Roberto Vitillo.
    -  Designing Distributed Systems by Brendan Burns (Free Ebook)
8. System Design
9. General
    - The Pragmatic Programmer by David Thomas and Andrew Hunt.
    - Modern Software Engineering by David Farley
    - Code Complete: A Practical Handbook of Software Construction by Steve McConnell
    - Software Engineering at Google - by Titus Winters, Tom Manshreck, and Hyrum Wright (Free Ebook
10. Leadership
    - The Five Dysfunctions of a Team by Patrick Lencioni
    - Drive by Daniel Pink
    -  The Making of a Manager by Julie Zhuo
11. Good practices
    - Clean Code by Uncle Bob Martin.
    - Head First Design Patterns by Eric Freeman.
    - Refactoring by Martin Fowler.
12. Testing
    - Growing OO Software by Tests by Steve Freeman.
    - Unit Testing Principles, Practices, and Patterns by Vladimir Khorikov
    - The Art of Unit Testing by Roy Osherove.
    - TDD by Example by Kent Beck
13. Software Architecture
    -  Fundamentals Of Software Architecture by Mark Richards and Neil Ford.
    - Clean Architecture by Uncle Bob Martin.
    - Software Architecture The Hard Parts by Neal Ford, Mark Richards, Pramod Sadalage, and Zhamak Dehghani.
    -  Domain Driven Design Quickly by Abel Avram and Floyd Marinescu (Free Ebook).
    - A Philosophy of Software Design by John Ousterhout.
14. DevOps
    - DevOps Handbook by Gene Kim, Patrick Debois, John Willis, and Jez Humble.
    - Continuous Delivery by Jez Humble and David Farley.
    -  Accelerate by Nicole Forsgren, Jez Humble, and Gene Kim.
15. Machine Learning
    - The Hundred-Page Machine Learning Book by Andriy Burkov
    - Designing Machine Learning Systems by Chip Huyen
16. Personal Development 
    - How to Win Friends & Influence People by Dale Carnegie
    - Deep Work by Cal Newport
    - The 7 Habits of Highly Effective People by Stephen R. Covey
    - Atomic Habits by James Clear
